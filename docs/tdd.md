# Техническое задание: Генератор отзывов

## 1. Общее описание
Система для генерации уникальных, корректных отзывов на товары.
Критерии: Уникальность, Логичность, Соответствие тех. характеристикам, Равномерное распределение по датам.

## 2. Входные данные
- Поисковые запросы (название + модель)
- URL (опционально). Если нет -> Автопоиск.
- Количество отзывов (Явное или расчитанное по периоду).
- Валидация: Сумма отзывов должна совпадать с общим кол-вом.

## 3. Сбор данных (Perplexity)
- Название, Описание, Характеристики.
- Существующие отзывы (для проверки уникальности).

## 4. Генерация (Mistral -> DeepSeek)
- **Этап 1 (Mistral):** Предварительная генерация.
- **Этап 2 (DeepSeek):** Верификация и доработка.
  - Проверка уникальности.
  - Логичность, тон.
  - Техническая точность.

## 5. Структура отзыва
- Имя (случайное)
- Дата (равномерно в периоде)
- Оценка (1-5)
- Плюсы
- Минусы
- Общее впечатление (Текст)
- URL товара
- Статус "Использовано"

## 6. Интерфейс
- Таблица с редактируемыми полями.
- Кнопка "Копировать".
- Чекбокс "Использовано".
- Блокировка выгрузки, пока все не использованы.

## 7. Технические требования
- Проверка уникальности (Cosine similarity / Jaccard).
- Производительность до 200 отзывов.
